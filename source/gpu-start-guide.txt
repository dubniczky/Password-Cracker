Visual Studio

1. Run the following bat file: ..\oclpack\mount_t.bat
2. Open the Solution file: .\PassHash.sln
3. Select x86 Debug mode and run the program

Other IDE/Compiler

1. Set ../oclpack/ as an include library
2. Compile the ./SHA256GPU/main.cpp
3. Make sure that the *.kernel.cl files are in the same directory as the executable
